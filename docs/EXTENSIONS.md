# 🔌 Extensii Recomandate (Recommended Extensions)

Pentru a avea o experiență perfectă când scrii cod în **Antigravity** (care se bazează pe nucleul VS Code în spate), îți recomand să instalezi următoarele extensii esențiale. (În cazul în care extensiile nu au fost deja instalate automat cu setările corecte)

### Cum să le instalezi:

1. Mergi în meniul din stânga la pictograma cu cele 4 pătrățele (Extensions) sau apasă `Ctrl+Shift+X` (`Cmd+Shift+X` pe Mac)
2. Caută numele extensiei exact cum e scris în paranteză și dă click pe **Install**.

---

### 1. ESLint (`dbaeumer.vscode-eslint`)

Acesta este "Poliția Codului". Îți subliniază cu roșu când ai uitat o virgulă, un bracket sau când o variabilă nu a fost folosită nicăieri. Este vital să îl ai pentru a prinde erorile urâte înainte să te chinuiești să dai refresh la site.

### 2. Tailwind CSS IntelliSense (`bradlc.vscode-tailwindcss`)

Fără el ești pe nevăzute când stilizezi! Când scrii `classN...`, această extensie îți auto-scrie `className="..."` și mai important: când începi să scrii o clasă Tailwind (ex: `bg-red`), îți arată o listă cu toate clasele disponibile și culoarea exactă lângă ele pe un pătrățel mic.

### 3. Prettier - Code formatter (`esbenp.prettier-vscode`)

Îți aranjează automat codul pe linii drepte și îl face perfect lizibil. Instalează-l și apasă `Alt+Shift+F` pe orice fișier dezordonat ca să-l facă să arate curat, cu alinieri perfecte. **Atenție:** Mergi în setările editorului (`Ctrl + ,` / `Cmd + ,`), caută **"Format On Save"** și bifează opțiunea. Apoi setează **"Default Formatter"** la `esbenp.prettier-vscode` pentru ca la fiecare `Ctrl+S` să se asigure alinierea automată.

### 4. VSCode PDF (`tomoki1207.pdf`)

Când deschizi un fișier `.pdf` (cum ar fi pitch-ul vostru de Hackathon, detaliile pe care le-ați primit, sau vreo documentație uriașă de referință) în interiorul aplicației, această extensie te lasă să-l citești direct în tab-urile din Antigravity. Nu mai trebuie să faci Alt+Tab ca să mergi la Chrome și Google Drive!

# 🔌 Recommended Extensions for Antigravity

To ensure a seamless coding experience in **Antigravity** (which operates like VS Code under the hood), I highly recommend installing the following essential extensions.

### How to install them:

1. Open the Extensions view by clicking the four-square icon on the left sidebar or by pressing `Ctrl+Shift+X` (`Cmd+Shift+X` on Mac).
2. Search for the exact extension name (in parentheses below) and click **Install**.

---

### 1. ESLint (`dbaeumer.vscode-eslint`)

The "Code Police". It highlights code errors, missing commas, missing brackets, or unused variables with red squiggly lines. It's vital for catching frustrating bugs before testing them in the browser.

### 2. Tailwind CSS IntelliSense (`bradlc.vscode-tailwindcss`)

You are flying blind without this when styling! When you start typing a Tailwind class like `bg-red`, it provides an auto-complete drop-down list of all available classes and displays the exact color preview in a tiny square next to it.

### 3. Prettier - Code formatter (`esbenp.prettier-vscode`)

This automatically formats your code, making it instantly neat and readable. Once installed, simply press `Alt+Shift+F` in any messy file to organize it with perfect indentation. **Pro Tip:** Go into your editor's Settings (`Ctrl + ,` / `Cmd + ,`), search for **"Format On Save"** and toggle it on. Then set your **"Default Formatter"** to `esbenp.prettier-vscode` to ensure your code snap to style automatically whenever you hit `Ctrl+S`.

### 4. VSCode PDF (`tomoki1207.pdf`)

If you have a `.pdf` file (like your Hackathon pitch, constraints, or a reference document) in your folder, this extension allows you to open and read it directly inside Antigravity's tabs. This way, you don't have to constantly Alt-Tab back to Chrome or your PDF viewer!
